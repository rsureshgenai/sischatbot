# SIS AI Assistant 🤖

A conversational AI chatbot built using Streamlit for SIS International Recruiters.

## 🚀 Features
- Employer & Candidate flows
- Step-by-step chat interaction
- Mobile responsive UI
- Handles any input (no strict keywords)
- Clean chatbot interface

## 📦 Installation

```bash
pip install -r requirements.txt
```

## ▶️ Run App

```bash
streamlit run sis_ai.py
```

## 📱 Use Case

- Website chatbot
- Recruitment automation
- Lead conversion tool
- Customer interaction bot
